### 내 답
<pre><code>
#define _CRT_SECURE_NO_WARNINGS
#include stdio.h
#include string.h

int main(void) {
	char a[10],b[10],max[10];
	scanf("%s %s", a, b);
	for (int i = 2; i >= 0; i--) {
		if (a[i] > b[i]) {
			for(int j=2;j>=0;j--){
				max[j] = a[j];
				printf("%c", max[j]);
			}
			break;
		}
		if (a[i] < b[i]) {
			for (int j = 2; j >= 0; j--) {
				max[j] = b[j];
				printf("%c", max[j]);
			}
			break;
		}
	}
}

#### 느낀점(배운점)
1.문자열을 받아 3자리 수이니 1의자리부터 비교
2.높은지를계속 배열로 확인한다음
3.max값에 그위치를 넣고 출력
