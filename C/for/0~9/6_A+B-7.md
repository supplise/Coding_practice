### 내 답
<pre><code>
#define _CRT_SECURE_NO_WARNINGS
#include stdio.h

int main(void) {
	int count,a,b;
	scanf("%d", &count);
	for (int i = 0; i <count ; i++) {
		scanf("%d %d", &a, &b);
		printf("Case #%d: %d\n", i + 1, a + b);
	}
}
</code></pre>


#### 느낀점(배운점)
순서는 i에 +1를 해줘었고 남은 값출력은 A+B를 받아서 출력
